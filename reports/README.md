# Reports

This folder should contain the two required reports:

```text
reports/
├── Manager_Report.pdf
└── Teammate_Technical_Report.pdf
```

The manager report should be short, business-oriented, and understandable for a non-technical manager.

The teammate report should be more technical and should discuss preprocessing, augmentation, models, training setup, dataset issues, error analysis, model results, external test set performance, and future improvements.
